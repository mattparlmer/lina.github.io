## Git Workflow
* Before initializing, make sure you are in the right directory!!!
* git init (initialize)

* git remote add origin https://github.com/...
* git status
* git add .
* git commit -m "commit message"
* git push origin master

## UNIX command lines
* pwd - show me where I am in my directories
* ls - list files
* cd - change directory
